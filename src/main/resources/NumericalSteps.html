Conversions

Given a Binary number how do we convert to Hexadecimal,
We can use the following steps:

Step 1: Grouping

We start by creating groups of 4 from the binary number
example:
11001010 => 1100 1010


Step 2: Convert to Decimal

To convert the grouped binary numbers to decimal we multiply all the values by base of 2
and the multiplicity of the value is the position it's located.
1100 1010 = [(2*1)3+(2*1)2+(2*0)1+(2*0)0 ] [(2*1)3+ (2*0)2+(2*1)1+(2*0)0] => [12][10]


Step 3: Combine the resulting value to Hexadecimal

the last step is converting the decimal values from the previous step to hexadecimals like this
[12][10] => CA
