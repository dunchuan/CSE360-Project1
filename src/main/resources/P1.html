Number Systems and Conversions

Binary Numbers

In mathematics and digital electronics, a binary number is a number expressed in the binary
numeral system or base-2 numeral system which represents numeric values using two different symbols:
typically 0 (zero) and 1 (one). The base-2 system is a positional notation with a radix of 2.
